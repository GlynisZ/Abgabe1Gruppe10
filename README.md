# Dies ist das Repository zur ersten Abgabe für das Fach Software Engineering an der Hochschule Karlsruhe im Wintersemester 20/21.
In diesem Repository befindet sich ein Projekt in Typescript mit einer GraphQL-Schnittstelle und implementiert mit Teilen aus dem MEAN Umfeld. Es ist ein beispielhaftes Projekt mit Tests, SonarQube und Jenkins Integration.
Im Ordner doc finden sie ein Entwicklerhandbuch für Asciidoctor mit Diagrammen erstellt durch PlantUML.
